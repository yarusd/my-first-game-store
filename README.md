# 🎮 My First Game Store

Welcome to **My First Game Store** — a Python project with several small games, created as a playground and used for automated testing with tools like Playwright.

## 🧠 What’s Inside
This repository includes:
✔ Multiple Python game files  
✔ Playground for test automation practice  
✔ Examples of UI logic that can be tested using Playwright

## 🕹️ Games Included
- Rock-Paper-Scissors  
- Tic-Tac-Toe  
- Guess the Word  
- Battle Game  
… and more!
